## Getting Started

To get a local copy up and running follow these simple steps:

1. Clone the repo
   ```sh
   git clone https://github.com/zerone0x/trine.git
   ```
2. Install bun packages
   ```sh
   bun install
   ```
3. Start the application
   ```sh
   bun run dev
   ```
